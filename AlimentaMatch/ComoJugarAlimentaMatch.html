Reglas Generales

AlimentaMatch es un juego de puzzle donde el objetivo principal es combinar 3 o más elementos del mismo tipo para eliminarlos del tablero y ganar puntos:

  • Combina 3 o más elementos iguales para eliminarlos.
  • Puedes combinar elementos en horizontal o vertical, pero no en diagonal.
  • Completa los objetivos antes de quedarte sin tiempo.
  • Utiliza potenciadores y elementos especiales estratégicamente para superar niveles difíciles.

Cómo Jugar

  1. Abre la aplicación o el juego AlimentaMatch en tu dispositivo.
  2. Selecciona un nivel para jugar.
  3. Observa el objetivo del nivel y los movimientos disponibles.
  4. Toca y arrastra los elementos del tablero para combinar 3 o más del mismo tipo.

Frutas

Las frutas son una excelente fuente de vitaminas y minerales esenciales para la salud. Son ricas en fibra, antioxidantes y agua.

Algunas frutas comunes incluyen manzanas, naranjas, plátanos, fresas, etc.

Verduras

Las verduras son ricas en nutrientes como vitaminas, minerales y fibra. Son importantes para mantener una dieta equilibrada y saludable.

Algunas verduras populares son espinacas, zanahorias, brócoli, etc.

Leguminosas

Las leguminosas son una excelente fuente de proteínas, fibra y carbohidratos complejos. Son importantes para una dieta vegetariana o vegana equilibrada.

Algunas leguminosas comunes son los frijoles, las lentejas, los garbanzos, etc.

Tubérculos

Los tubérculos son una buena fuente de carbohidratos y proporcionan energía duradera. Son versátiles y se pueden preparar de varias formas.

Algunos tubérculos populares son las papas, las batatas, las zanahorias, etc.
